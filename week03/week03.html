JSON Parse, Stringify



JSON string:
var text = '{ "employees" : [' +
'{ "firstName":"Josh" , "lastName":"Vilchis", "gender":"male", "age":22 },' +
'{ "firstName":"Anna" , "lastName":"Smith", "gender":"female", "age":22 },' +
'{ "firstName":"Peter" , "lastName":"Jones", "gender":"male", "age":1 } ]}';

We will then take this string and parse it into an object.
press the parse button above. Below will show what the object looks like after
being parsed

Notice that all we see from the output, is that it is now an object.

In order to get the information from inside of the object, consider it almost like a class
in this instance; we have a JSON string turned prased into an object.
Now if we take the object, and access the "properties", we can see what it actually is.


Let's take a look at Josh Vilchis. To do this, we simply take our object variable:
obj.employees[1].firstName + " " + obj.employees[1].lastName + " is a "
obj.employees[1].gender + " and is " + obj.employees[1].age + " years old."
